# Project-O

## Instruction

- Clone repository
  ```
  git clone git@github.com:Arino4kaMyr/Project-O.git
  ```
- Write program using Language O in directory src/main/resources
- In file stc/main/kotlin/constants/MainConstants.kt in TEST_FILE write name of your Program O file
- Download jasmin and add at directory jasmin/generated file jasmin.jar
- If you have a Mac, in terminal from root of repository write:
  - ```
    brew install jasmin
    ```
  - ```
    cp /opt/homebrew/Cellar/jasmin/2.4_2/libexec/jasmin.jar jasmin/generated/jasmin.jar
    ```
- Just run Main.kt and at dir jamin/generated you will get generated files and result at console!
